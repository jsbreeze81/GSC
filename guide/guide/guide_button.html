버튼그룹

  • 기본 우측정렬
  • 좌측 정렬 : 'on_left' 클래스 사용
  • 버튼 간격 : 'gap' 클래스 사용 (좌측간격 추가)
목록 이전글 다음글 
            <!-- 버튼 -->
            <div class="btnArea">
                <a href="#" class="btnSt is-primary">목록</a>
                <a href="#" class="btnSt is-secondary on_left">이전글</a>
                <a href="#" class="btnSt is-secondary on_left gap">다음글</a>
            </div>
            <!-- //버튼 -->
