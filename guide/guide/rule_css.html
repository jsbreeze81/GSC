CSS 기본 규칙

주석표기

  • 코드 가독성을 높기이 위해 주석을 사용한다.
  • 종료주석은 사용하지 않는다.
            /* 공통 스타일 */
            .common{...}

z-index

구분 기본값 증감단위
일반 요소 0 1
레이아웃 요소 100 10
팝업 요소 1000 100

웹폰트

  • 웹폰트는 'common.css'에 선언한다.
  • 사용폰트 : Noto Sans KR, Noto Serif KR
  • 직접 선언
            /* font - noto sans kr */
            @font-face {
                font-family: 'Noto Sans KR';
                font-style: normal;
                font-weight: 200;
                src: url(../font/NotoSans-Light.woff2) format('woff2'),
                    url(../font/NotoSans-Light.woff) format('woff'),
                    url(../font/NotoSans-Light.otf) format('opentype')
            }
            @font-face {
                font-family: 'Noto Sans KR';
                font-style: normal;
                font-weight: 300;
                src: url(../font/NotoSans-DemiLight.woff2) format('woff2'),

해상도 분기

  • 미디어쿼리를 이용해 분기한다.
  • 상황에 맞게 해상도 설정
  • PC 화면부터 작업하는 경우 기본 분기
            @media screen and (max-width:1240px){
                헤더, 푸터 분기
            }
