함수

코드 들여쓰기

  • 1탭 들여쓰기 하며, 1탭의 크기는 공백 4칸으로 설정한다.

주석표기

  • 코드 가독성을 높기이 위해 주석을 사용한다.
  • 종료 주석의 경우 '//'를 붙여 구분해 준다.
            <!-- 시작 -->
            <div>
            </div>
            <!-- //종료 -->

수정내용 표기

  • 추가/수정/삭제등 변경이 있을 경우 해당내용을 주석으로 남긴다.(날짜, 수정내용, 이슈)
  • 여러줄 수정일 경우 종료 주석을 사용한다.
            <!-- 190220 텍스트수정 -->

기능설명

  • 상태변화나 기능구현에 사용하는 클래스명은 주석으로 내용을 설명한다.
            <!-- 메뉴 선택했을 때 스타일 변경 : is-selected 클래스 추가 -->
            <div class="guide_is-selected">
            메뉴명
            </div>
